# Predicting access to elite higher education in England

Which local authorities are most likely to miss out on top-third HE
destinations — and why?

## Status
Work in progress — Phase 1 (Data Understanding) complete.

## Data source
DfE: Progression to Higher Education or Training, 2023/24 release.
Open Government Licence v3.0.

## Setup
pip install -r requirements.txt

## Project structure
notebooks/   — Jupyter notebooks, one per phase
src/         — Reusable Python functions
data/raw/    — Raw data (not tracked by Git — see .gitignore)
models/      — Saved model artefacts
outputs/     — Charts and visualisations
